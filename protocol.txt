AVA local protocol

   postgresql-x64-16

2. Start AVA services:
   cd D:\AVA_Project
   .\ava_server_control.ps1 start

3. Backend:
   cd D:\AVA_Project\NodeBackend
   npm install
   npm start

4. Flutter:
   cd D:\AVA_Project\Flutter
   .\flutter_local.cmd pub get
   .\flutter_local.cmd analyze
