1. Docker Desktop 무료 사용 기준

Docker Personal 기준으로 다음 조건에서는 Docker Desktop을 무료로 사용할 수 있습니다.

- 개인 사용
- 교육 목적
- 비상업 오픈소스
- 직원 250명 미만 AND 연매출 1천만 달러 미만 소규모 사업자

위 조건을 벗어나는 대기업/정부기관/일부 상업 사용은 Docker 유료 구독이 필요합니다.

2. Spring Boot 백엔드 실행

```powershell
cd D:\AVA_Project\SpringBoot

$env:JAVA_HOME='C:\Program Files\Microsoft\jdk-21.0.11.10-hotspot'
$env:Path="$env:JAVA_HOME\bin;$env:Path"

docker compose up -d
.\gradlew.bat bootRun
```

Spring Boot Docker Compose Support가 `compose.yaml`의 개발용 서비스를 사용합니다.

- PostgreSQL: `localhost:5432`
- MongoDB: `localhost:27017`
- Redis: `localhost:6379`

3. Flutter Windows 앱 실행

