AVA run protocol

1. Docker must not be used for AVA development or verification.
2. Do not add compose files or container runtime dependencies.
3. Start required databases and services as local/native Windows services,
   standalone binaries, or external endpoints.
4. AZOOM SFU/media:

   cd D:\AVA_Project\SpringBoot
   .\start_azoom_sfu.ps1

   External access requires TCP 7880, TCP 7881, and UDP 50000-50100 forwarded
   to the server PC.

5. Spring Boot:

   cd D:\AVA_Project\SpringBoot
   .\gradlew.bat test
   .\start_backend_with_azoom_sfu.ps1

6. Flutter Windows:

   cd D:\AVA_Project\Flutter
   .\run_windows.cmd

7. Seeded system account:

   admin@ava.admin / Ava1234! (SUPERUSER)

8. Main backend URL:

   http://112.166.136.198:8080
