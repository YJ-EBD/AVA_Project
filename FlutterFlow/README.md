# FlutterFlow Workspace

This folder is intentionally light. It is for notes and reviewed FlutterFlow-derived assets only.

FlutterFlow's GitHub integration should push generated code to the `flutterflow` branch, not directly into `main`.

Use `tooling/flutterflow/pull-flutterflow.ps1` to fetch the generated branch into `.flutterflow-worktree` for review.
